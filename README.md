# Scratch3教材

学生が制作したScratchの教材を、PowerPoint形式で公開します。

教材では、Scratch 3を用いたプログラミング入門の他に、ビデオモーションセンサーによるAR、Makey Makeyの利用なども紹介しています。

## 教材の作者

八戸工業大学工学部システム情報工学科 令和元年度小久保温研究室所属 佐藤雄也、山白康平

## この教材に関する連絡先
